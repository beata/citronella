_loadModule(string $module, boolean|array $createObj, string $appId) : void|\Module

Parameters

string $module

The name of the module to be loaded.
boolean|array $createObj

Whether or not to return the view model instance. If is array, each item will be the member attribute of the view model.
string $appId

Load model under $appId directory. If not set, current App::$id would be used.
